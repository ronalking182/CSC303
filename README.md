# CSC303

This repository hosts multiple assignments via GitHub Pages using the `docs/` folder.

## Structure
- `docs/index.html` — Landing page linking to each assignment
- `docs/assignment_01/` — Header & Footer responsive demo
- Add more assignments under `docs/assignment_##/`

## Hosting
1. Go to GitHub → Repo `Settings` → `Pages`
2. Set Source: `Deploy from a branch`
3. Branch: `main` and Folder: `/docs`
4. Your site will be available at:
   - `https://<username>.github.io/CSC303/` (landing page)
   - `https://<username>.github.io/CSC303/assignment_01/`

## Add a new assignment
- Create a folder: `docs/assignment_02/`
- Place `index.html`, `app.css` inside
- Link it from `docs/index.html`
- Push to `main`, Pages deploys automatically.
